# FVG Quality Analysis - 15-Minute Data (2018-2025)

## Overview

This document presents the Fair Value Gap (FVG) quality analysis for 15-minute trading data at the 8:30 AM candle from 2018 to 2025.

### Analysis Methodology

- **FVG Detection**: Based on three candles (8:15, 8:30, 8:45)
- **Quality Assessment**: Checks if price returns to FVG zone within 5 candles after FVG closes (candles 9:00-10:00)
- **Good FVG**: Price does NOT return to the FVG zone within 5 candles
- **Bad FVG**: Price DOES return to the FVG zone within 5 candles

## Key Insights

1. **Overall Pattern**: 15-minute timeframe shows the best good FVG rate (41.53%) across all timeframes analyzed.

3. **Worst Year**: 2023 (33.01% good FVG) - showing notable market regime change.

4. **Remarkable 2021**: Only instance across all timeframes/years where good FVG nearly equals bad FVG.

5. **Recovery Pattern**: 2024-2025 maintaining stable ~42-43% good FVG rate after 2023's decline.

6. **Timeframe Advantage**: 15-minute shows best overall reliability for FVG holds among all three timeframes.
